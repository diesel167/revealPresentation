An open-source programming language developed and maintained by Microsoft. It is a strict syntactical superset of JavaScript, and adds optional static typing to the language.
TS - Compiled (transpiled) vs JS - Interpretable
Strict typing
Static typing
Explicit or implicit typing

## Benefits:

-More reliable code

-Scope of class properties

-Fewer tests

-Commit-security

## Limitations:

-Difficult using of libraries&frameworks

-Not popular

-Longer development
